## README
# IDSOAggregationProblem
[![DOI](https://zenodo.org/badge/1094248281.svg)](https://doi.org/10.5281/zenodo.17582537)

The paper cited below [1], consitutes the main documentation of this repository. The methodology employed to build this repository is explained in detail in the paper. This repository contains all the relevant code, data files, scripts and documentation of this paper. 

The USAGE.pdf file provides a detailed overview of the repository and also a way to run the IDSO's Aggregation Problem with condition used in the original paper.

The repository is licenced under BSD 3-Clause ("New" or "Revised") License and the material is being released as an open source form.


[1] Sharma, S., Battula, S. and Singh, S.N., 2025. IDSO-Managed Bid-Based Transactive Distribution Systems Design for DER Participation in Wholesale Markets While Preserving TD Interactions. arXiv preprint arXiv:2508.08187. Available: https://arxiv.org/abs/2508.08187
