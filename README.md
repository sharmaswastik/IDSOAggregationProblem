# README
## IDSO's Aggregation Problem for a Bid-based Transactive Energy System Design of Distribution System.

[![DOI](https://zenodo.org/badge/1094248281.svg)](https://doi.org/10.5281/zenodo.17582537)

The paper cited below [1] serves as the main documentation for this repository. The methodology employed to build this repository is explained in detail in the paper. This repository contains all the relevant code, data files, scripts, and documentation of this paper. 

The USAGE.pdf file provides a detailed overview of the repository and also outlines a method for running the IDSO's Aggregation Problem with the conditions used in the original paper.

The repository is licensed under the BSD 3-Clause ("New" or "Revised") License, and the material is being released as open source.
